Este proyecto es una API REST construida con Node.js, Express y MongoDB, diseñada para gestionar usuarios del sistema del jardín "El Duende". Permite crear, consultar, actualizar y eliminar usuarios, además de autenticarlos mediante login con JWT.

Tecnologías utilizadas
- Node.js + Express
- MongoDB + Mongoose
- Postman (para testing)
- JWT (para autenticación)
- Bcrypt (para encriptar contraseñas)
- Dotenv (para variables de entorno)
